# Code Simplifier (서브에이전트)

작업 완료된 코드를 리뷰하고 단순화합니다.

## 리뷰 대상

$ARGUMENTS

대상이 지정되지 않으면 최근 수정된 파일들을 검토합니다:
```bash
git diff --name-only HEAD~1
```

## 단순화 체크리스트

### 1. 불필요한 복잡성 제거

- [ ] 사용하지 않는 import 제거
- [ ] 사용하지 않는 변수/함수 제거
- [ ] 중복 코드 통합
- [ ] 과도한 중첩 구조 평탄화

### 2. 가독성 개선

- [ ] 함수가 한 가지 일만 하는지 확인
- [ ] 변수명이 의도를 명확히 표현하는지 확인
- [ ] 복잡한 조건문 단순화
- [ ] 매직 넘버를 상수로 추출

### 3. Vue 컴포넌트 최적화

- [ ] 컴포넌트가 단일 책임 원칙 준수
- [ ] Props/Emits 최소화
- [ ] computed vs method 적절성
- [ ] 불필요한 reactive 래핑 제거

### 4. TypeScript 개선

- [ ] 타입 추론 가능한 곳의 불필요한 타입 명시 제거
- [ ] 유니온 타입 대신 enum/const 사용 검토
- [ ] 제네릭 과용 여부

## 리뷰 결과 형식

```markdown
## 파일: [파일 경로]

### 발견된 이슈
1. [이슈 설명]
   - 현재: [코드 스니펫]
   - 개선: [개선된 코드]

### 권장 변경사항
- [ ] [변경 1]
- [ ] [변경 2]
```

## 주의사항

- 기능 변경 없이 리팩토링만 수행
- 테스트가 있다면 테스트 통과 확인
- 팀 컨벤션 유지 (CLAUDE.md 참조)
